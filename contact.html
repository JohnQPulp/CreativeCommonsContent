Contacting John Q. Pulp

You can reach out to me in these ways:

Github

If you want to leave a public comment, you can open an issue on one of the site's Github repos.

For general things like website code issues or new book suggestions or death threats, use the site's main repository: https://github.com/JohnQPulp/PublicDomainPulp. Whereas if you a book-specific comment or suggestion about one of the existing VNs, you can leave the issue on that specific VN's repo (e.g., https://github.com/JohnQPulp/PuddnheadWilson for Pudd'nhead Wilson).
